# Chatbot Multimodelo con Fallback Automático

Chatbot inteligente en Python que integra múltiples proveedores de IA con un sistema de fallback en cascada (OpenAI → Anthropic → Google Gemini). Mantiene el historial de conversación y utiliza streaming.

## Uso
1. Instala dependencias: `pip install -r requirements.txt`
2. Coloca tus API keys en el archivo `.env`
3. Inicia el bot: `python main.py`
